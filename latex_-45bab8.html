Latex_环境配置

  • 这个页面是为了 LaTeX 环境配置，Linux 和 Windows.

最常用的的也是

  • 首先，你需要安装TeX Live，它是一个开源的 LaTeX 发行版本，包括了数百种的 LaTeX 工具和模板。
  • 然后，你需要安装 TeX\TeXTE​X 编辑器，比如Emacs来编辑 .tex 文件，或者TeXstudio来编辑 .tex 文件和 .pdf 文件。
  • 最后，你需要安装一个可以编译 LaTeX 文件的编译器，比如 TeXLive 或 MiKTeX。

如何在VScode中配置latex

  • 在VSCode中，你需要安装 LaTeX-Workshop 插件，它可以帮助你编辑、编译和预览 LaTeX 文件，并且支持 Windows 和 Linux。
  • 当你安装完 LaTeX-Workshop 插件后，你需要在VSCode的设置中指定 LaTeX 编译器的路径，以便VSCode能够正确的编译你的文档。
  • 然后，你可以在VSCode中使用快捷键来编译文档，或者使用编译按钮在VSCode侧边栏中编译文档。
